News Explorer — Frontend

News Explorer es una aplicación web desarrollada con React y Vite que permite a los usuarios buscar noticias en tiempo real utilizando la API de NewsAPI, visualizar resultados y simular el guardado de artículos cuando el usuario está autenticado.

Este proyecto corresponde a la implementación exclusiva del Front-End, siguiendo el diseño proporcionado en Figma y cumpliendo los criterios de evaluación de TripleTen.

🛠 Tecnologías utilizadas

React (Hooks)
Vite
React Router DOM
CSS (BEM)
API externa: newsapi.org
Proxy de TripleTen: nomoreparties.co
LocalStorage

🚀 Funcionalidades principales

🔍 Búsqueda de noticias por palabra clave
⏳ Preloader durante solicitudes a la API
❌ Manejo de errores de la API
📭 Mensaje cuando no hay resultados
📰 Renderizado de tarjetas de noticias
➕ Botón “Mostrar más” (3 tarjetas por carga)
💾 Simulación de guardado de artículos
🔐 Simulación de autenticación (isLoggedIn)
💬 Tooltip cuando el usuario no está logueado
📱 Diseño responsive (desktop, tablet y mobile)
💾 Persistencia de búsqueda con LocalStorage
